Windows

Metro 4 provides special classes to create window elements and manipulate them.

Create window

To create window you can use next classes: .window, .window-caption, .window-content.

Caption

To create caption add container with class .window-caption. Caption can contains: icon, title and buttons.

Icon

To create window icon you can use icon font or image. Add element with class .icon: 


                    <div class="window">
                        <div class="window-caption">
                            <span class="icon mif-rocket"></span>
                        </div>
                    </div>

                    <div class="window">
                        <div class="window-caption">
                            <img class="icon" src="images/location.png">
                        </div>
                    </div>

Title

To create title add <span> with class .title to caption:

The rocket
Target 

                    <div class="window">
                        <div class="window-caption">
                            <span class="title">Target</span>
                        </div>
                    </div>

Buttons

You can create window buttons such as: close, min and max and styles as default Windows window buttons. To create buttons first-off you must create buttons container with class buttons. Then you can create inside the container window system buttons.

The rocket 

                    <div class="window">
                        <div class="window-caption">
                            <span class="icon mif-rocket"></span>
                            <span class="title">The rocket</span>
                            <div class="buttons">
                                <span class="btn-min"></span>
                                <span class="btn-max"></span>
                                <span class="btn-close"></span>
                            </div>
                        </div>
                    </div>

Role window

If you add attribute data-role="window" to element, you activate Metro 4 handler for window.

The rocket
This is a element with full window structure transformed to window. 

                    <div class="window win-shadow" data-role="window">
                        <div class="window-caption">
                            <span class="icon mif-rocket"></span>
                            <span class="title">The rocket</span>
                            <div class="buttons">
                                <span class="btn-min"></span>
                                <span class="btn-max"></span>
                                <span class="btn-close"></span>
                            </div>
                        </div>

                        <div class="window-content">
                            <div class="p-2">
                                This is a element with full window structure transformed to window.
                            </div>
                        </div>
                    </div>

You can transform to window any block element:

This is a simple DIV transformed to window with data-role. 

                    <div class="p-2" data-role="window">
                        This is a simple DIV transformed to window with data-role.
                    </div>

To set title and icon add attributes data-icon="..." and data-title="...".

This is a simple DIV transformed to window with data-role.

Customize

You can change default colors for caption and content. To set it use attributes data-cls-caption="..." and data-cls-content="...".

This is a simple DIV transformed to window with data-role. 

                    <div class="p-2"
                         data-role="window"
                         data-icon="<span class='mif-rocket'></spam>"
                         data-title="The rocket"
                         data-cls-caption="bg-red"
                         data-cls-content="bg-dark fg-yellow">
                        This is a simple DIV transformed to window with data-role.
                    </div>

Draggable window

To create draggable window add attribute data-role="draggable" to your window. About draggable you can read in this article.

The rocket
Drag this window on the caption. 

                    <div class="window pos-absolute"
                         style="width: 220px; height: 160px;"
                         data-role="window, draggable"
                         data-drag-element=".window-caption .title">
                        <div class="window-caption">
                            <span class="icon mif-rocket"></span>
                            <span class="title">The rocket</span>
                            <div class="buttons">
                                <span class="btn-min"></span>
                                <span class="btn-max"></span>
                                <span class="btn-close"></span>
                            </div>
                        </div>

                        <div class="window-content">
                            <div class="p-2">
                                Drag this window on the caption.
                            </div>
                        </div>
                    </div>

Resizeable window

To make window as resizable add attribute data-resizable="true" to your window.

The rocket
Drag this window on the caption. Resize window by click end move right-bottom window corner. 

                    <div class="window pos-absolute"
                            style="width: 220px; height: 160px;"
                            data-role="window, draggable"
                            data-drag-element=".window-caption .title"
