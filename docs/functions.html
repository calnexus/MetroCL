Functions

Metro 4 implements several useful functions that are accessible through the special object.

About

All special utilities functions stored in $.Metro.utils object.

  • isUrl(val) - check string is url
  • isColor(val) - check string is hex color
  • secondsToTime(time) - return object {h, m, s}
  • hex2rgba(hex, alpha) - return rgba value for hex color
  • random(from, to) - return random value
  • isInt(v) - check value is integer
  • isFloat(v) - check value is float
  • uniqueId() - return unique value in format xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx
  • isTouchDevice() - check device is touchable
  • secondsToFormattedString(time) - return string value in format hh:mm:ss
  • callback(func) - execute function or code with return value
  • exec(func) - execute function or code with return value
  • isFunc(val) - if val is function return function object else return false
  • isJQueryObject(obj) - check value is jQuery object
  • addLocale(locale) - add locale at runtime
  • getLocaleNames() - return registered locales
  • elementInViewport(el) - check element visible in viewport
  • objectLength(obj) - return object length
  • percent(a, b, c) - return percent value
  • camelCase(val) - return string in camel case. my-value -> myValue
  • objectShift(obj) - remove first value from object
  • objectDelete(obj, key) - remove value from object
  • arrayDelete(array, key) - remove value from array
  • nvl(value, other) - return other if value is undefined or null else return value
  • github(repo, callback) - return github repo data to callback function
  • detectIE() - detect IE, Edge
  • md5(val) - calc md5 value for string
  • encodeURI(val) - fixed encode uri function
  • pageHeight() - return actual page height
