Preloaders

Preloaders indicators are visual indications of an app loading content.

About

Metro 4 provides classes and plugin to create preloaders. To create preloader you can add data-role="preloader" attribute to element and defin additional options.

Types

Metro 4 implements several preloaders, which you can use easily and simply. The preloader type defines with data-type=* attribute.

Ring


                    <div data-role="preloader" data-type="ring"></div>
                    <div data-role="preloader" data-type="ring" data-style="dark"></div>
                    <div data-role="preloader" data-type="ring" data-style="color"></div>

Metro


                    <div data-role="preloader" data-type="metro"></div>
                    <div data-role="preloader" data-type="metro" data-style="dark"></div>
                    <div data-role="preloader" data-type="metro" data-style="color"></div>

Square


                    <div data-role="preloader" data-type="square"></div>
                    <div data-role="preloader" data-type="square" data-style="dark"></div>
                    <div data-role="preloader" data-type="square" data-style="color"></div>
