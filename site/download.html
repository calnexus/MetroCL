Source files

Compile Metro UI CSS with your own asset pipeline by downloading our source Less, JavaScript, and documentation files. This option requires some additional tooling:

  • Less compiler for compiling your CSS
  • Autoprefixer for CSS vendor prefixing
  • Build tools for compiling js files

* Author use Grunt with required modules.

Metro CDN

Use Metro CDN to deliver cached version of Metro UI CSS with compiled CSS and JS to your project. 


                    <link rel="stylesheet" href="https://cdn.metroui.org.ua/metro-all.min.css">
                    <script href="https://cdn.metroui.org.ua/metro.min.js"></script>

* Metro CDN, provided for free by the folks at KeyCDN

Package managers

Pull in Metro UI CSS source files into nearly any project with some of the most popular package managers. No matter the package manager, Metro UI CSS will require a Less compiler and Autoprefixer for a setup that matches our official compiled versions.

npm

                    $ npm install metrouicss@latest
Bower

                    $ bower install metrouicss@latest
NuGet

                    PM> Install-Package metrouicss -Pre
